This is a mobile-friendly Android WebView hybrid app for a restaurant. The app includes the following features:

- Home page
- Menu page
- Contact page with a contact form
- Form validation
- Modern design

## How to Build the APK

1. Clone the repository or download the source code.
2. Open the project in Android Studio.
3. Build the app by clicking on the "Build" menu and selecting "Generate Signed Bundle / APK".
4. Follow the steps to create a new key store or use an existing one.
5. Select the "app" module and choose the "Release" build type.
6. Choose the output format as "APK".
7. Click "Finish" to generate the signed APK file.
8. The generated APK file can be found in the "app/build/outputs/apk/release" directory.

## How to Install the App

1. Transfer the generated APK file to your Android device.
2. On your device, go to "Settings" > "Security" and enable "Unknown sources" to allow installation of apps from sources other than the Google Play Store.
3. Locate the APK file on your device and tap on it to start the installation process.
4. Follow the on-screen instructions to complete the installation.

Enjoy using the Restaurant App!"}]}
